## 使用规则:
    双击index.html进入播放器页面进行操作
    输入搜索的歌曲后回车(或者点击搜索按钮)进行歌曲搜索
    点击歌曲名左侧的播放按钮(或者双击歌名框框)进行播放歌曲 同时右侧会有对应歌曲的评论
    如果有MV则会在歌曲名下出现"播放MV"字样 可以进行点击播放MV
    MV播放时可以对MV进行控制 点击MV周围区域则会关闭MV页面
